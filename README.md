## Getting Started

1. Clone this repository.

```bash
    git clone https://github.com/Synapso-Social-Media/synapso_ui.git
```

2. Install the necessary dependencies.

```bash
    cd synapso_ui
    npm install
```

3. Run the development server.

```bash
    npm run dev
```
